# 🍷 Wine Mood Pairing Curator 🍷

## 제품 서비스 설명
Wine Mood Pairing Curator는 사용자의 오늘 감정(편안한, 로맨틱한, 활기찬, 차분한, 우울한, 후끈한 中 1)을 선택하면 그 무드에 어울리는 와인을 추천해 주는 서비스입니다./n
레드, 화이트, 로제, 스파클링 총 4가지 카테고리로 구성되어 있으며, 개인 아카이브와 취향 프로필 관리 기능을 제공합니다.

---

## 주요 기능

### 1. 감정 기반 와인 큐레이션
* 오늘 나의 감정에 맞는 와인을 추천받을 수 있습니다.

### 2. 4대 카테고리 및 상세 정보 제공
* 레드, 화이트, 로제, 스파클링 총 4가지 카테고리의 와인을 분류하여 보여 줍니다.
* 각 와인별 원산지, 빈티지, 평점 정보를 확인할 수 있습니다.

### 3. 와인 아카이브 (보관함)
* 마음에 드는 와인은 하트 버튼을 눌러 개인 아카이브에 저장하고 언제든 다시 확인할 수 있습니다.

### 4. 마이 탭 (나만의 취향 프로필)
* 선호하는 와인 타입, 바디감, 산도, 맛 노트를 직접 설정하여 자신만의 와인 취향 프로필을 구축할 수 있습니다.
