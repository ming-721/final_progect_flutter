## 기말고사 과제물 - 어플리케이션 제작_플러터

**앱 이름:** Wine Mood Pairing Curator 

**제품·서비스 설명:**
Wine Mood Pairing Curator는 사용자의 오늘 감정(편안한, 로맨틱한, 활기찬, 차분한, 우울한, 후끈한)을 선택하면 그 무드에 어울리는 와인을 추천해주는 감성 기반 와인 큐레이션 앱입니다. 
레드·화이트·로제·스파클링 총 4가지 카테고리로 이루어져 있으며, 각 와인의 원산지, 빈티지, 평점, 푸드 페어링 정보를 확인할 수 있습니다. 
마음에 드는 와인은 하트를 눌러 아카이브에 저장할 수 있고, 마이 탭에서 선호하는 와인 타입·바디감·산도·맛 노트를 직접 설정해 나만의 취향 프로필을 만들 수 있습니다.
